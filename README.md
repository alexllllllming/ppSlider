ppSlider
========

A slider for presenting your powerpoint

## 功能

- 自定义标题导航
- 简单的切换效果
- 间隔自动播放
- 全屏播放

> 支持ie11,Firefox30, chrome27,Safari6.1

## 使用方法

1. 每个图片文件需要有统一的命名形式(前缀序号.文件类型),默认的前缀为`slide-`,开始序号为`0`,所以图片文件应该按照顺序以`slide-0.jpg`,`slide-1.jpg`...

2. 定义每个图片对应的标题

3. 调用ppSlider()
